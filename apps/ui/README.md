# GridFlex UI

Next.js command centre for the GridFlex Compute v2 demo.

## Run locally

```text
npm install
npm run dev
```

Open `http://localhost:3000` and use `/dashboard` for the demo page.

## Add a new page

Create a `page.tsx` file under `src/app/<route-name>/`.

Example: `src/app/jobs/page.tsx` becomes `/jobs`.
